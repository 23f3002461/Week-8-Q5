## Project Overview
This repository contains a Circle Packing visualization created using **RAWGraphs 1.0**.  
It represents an *investment portfolio allocation* across sectors and assets, demonstrating diversification and relative investment size.

## Files
- `chart.png` — Circle Packing chart (512×512px)
- `README.md` — Documentation and contact email

## Data Structure
The dataset includes:
- `sector` — Investment sector category
- `asset` — Individual asset or company
- `investment` — Value invested (numerical)

The visualization maps:
- **Hierarchy**: sector → asset  
- **Size**: investment amount  
- **Color**: sector  
- **Labels**: asset names

## Tools Used
- RAWGraphs (Circle Packing)
- CSV dataset (15–20 rows)
- Exported as PNG (512×512)
